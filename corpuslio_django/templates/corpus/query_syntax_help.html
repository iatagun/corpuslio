rocket_launch Quick Start

  1. Wrap attributes in square brackets: [pos="NOUN"]
  2. Use quotes for values: [word="test"]
  3. Combine with &: [word="test" & pos="NOUN"]
  4. Create sequences with spaces: [pos="ADJ"] [pos="NOUN"]
  5. Use regex patterns: [word=".*ing"]
